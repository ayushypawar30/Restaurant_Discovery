Restaurant Discovery Website

A responsive frontend web application that displays real-time restaurant data using an external API.
Built to practice API integration, search functionality, filtering logic, and modular UI architecture.

🚀 Features

🔍 Search restaurants by name

⭐ Filter restaurants based on ratings

🍜 Filter restaurants by cuisines

⚡ Real-time data fetching from external API

⏳ Proper loading state handling

❌ Error handling for failed API requests

📱 Fully responsive design

🛠 Tech Stack

Frontend: React.js

Language: JavaScript

Styling: CSS

API Integration: Swiggy API (Real-time data fetching)

🧠 What I Learned

Handling API calls using fetch / async-await

Managing component state in React

Implementing search and filter logic

Conditional rendering (loading & error states)

Writing clean and modular React components
